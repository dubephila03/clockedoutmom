This is a React + TypeScript single-page application (SPA) built with Vite. Specifically, it's a marketing/landing page website for a service called "ClockedOutMum".

Key characteristics:

Framework: React with TypeScript
Build tool: Vite
Styling: Tailwind CSS with a custom design system
UI Components: shadcn/ui-based component library
Animations: Framer Motion
Routing: React Router
State Management: React Query
Testing: Vitest (unit tests) and Playwright (e2e tests)
Linting: ESLint
The site features several landing page sections (Hero, About, Experiences, Community, Contact, Footer) and appears to be a community platform for mothers offering curated events and wellness experiences
